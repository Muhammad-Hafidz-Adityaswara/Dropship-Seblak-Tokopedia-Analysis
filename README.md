# **Dropshipping Seblak on Tokopedia : Analysis and Sales Strategy**
**Project Objective:**  
This project aims to analyze the sales potential of seblak products through a dropshipping scheme on the Tokopedia platform. By using web scraping techniques, sales data and public interest in seblak products will be analyzed to assist in designing effective sales strategies.

**Project Scope:**
1. Retrieving data from the Tokopedia website using web scraping techniques.
2. Analyze seblak product sales data, including sales trends, popularity and consumer preferences.
3. Identify the factors that influence sales of seblak products.
4. Design a sales strategy based on analysis findings to increase revenue from the seblak product dropshipping scheme.
